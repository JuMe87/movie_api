myFlix Application

This is the server-side component of a web application about movies.
This web application will provide users with information on different movies,
directors, and genres. User will be able to sign up, update their personal information,
and create a list of their favorite movies.

Here is a list of all essential features of this application:

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title (e.g., “Thriller”)
  • Return data about a director by name
  • Allow new users to register
  • Allow users to update their user info
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister

    • URL endpoints and request methods


    Client Request URL HTTP Method Request Body Data Format Response Body Data Format
    Return a list of ALL movies to the user /movies GET none A JSON object holding data about all the movies
    Return data (description, genre, director, image URL, whether it is featured or not) about a single movie by title to the user /movies/:title GET none A JSON object holding data about a single movie. Example:
    {
    title : "Titanic",
    genre: "Drama",
    director: "James Cameron"
    img_url: "xyz"
    }
    Return data about a genre (description) by name/title (e.g., “Thriller”) /genre/:genre_name GET none A JSON object holding data about a genre. Example:
    {
    genre_name : "Drama",
    description: "xyz"
    }
    Return data about a director (bio, birth year, death year) by name /director/:director_name GET none A JSON object holding data about a director. Example:
    {
    director_name : "James Cameron",
    birth_year: 1954,
    bio: "xyz"
    }
    Allow new users to register /users POST A JSON object holding the user information. Example:
    {
    "user_name": "Maxi123",
    "email": "mustermann@mail.de",
    "password": "123456!",
    "date of birth": "01.01.1990"
    }     		A success message indicating that the profile was successfully created.
    Allow users to update their user info (username) /users/:email/:user_name PUT none A success message indicating that the profile was successfully updated.
    Allow users to add a movie to their list of favorites /users/:email/favorites/:title POST none A success message indicating the name of the movie that has been successfully added to the list of favorites.
    Allow users to remove a movie from their list of favorites /users/:email/favorites/:title DELETE none A success message indicating the name of the movie that has been successfully removed from the list of favorites.
    Allow existing users to deregister /users/:email DELETE none A success message indicating that the profile was successfully deleted.
